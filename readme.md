# Password Generator Tool

## Description
The Password Generator Tool is a simple Python application that generates secure random passwords. It allows users to choose the password length and creates a password using uppercase letters, lowercase letters, numbers, and special characters.

## Features
- Generate secure random passwords
- User can choose the password length
- Includes uppercase and lowercase letters
- Includes numbers
- Includes special characters
- Simple command-line interface

## Technologies Used
- Python 3
- random module
- string module

## How to Run
1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the following command:
   ```
   python password_generator.py
   ```
4. Enter the desired password length.
5. The generated password will be displayed on the screen.
